## üìù How to Use

1. **Add images to this folder:**
   - Use high-quality jewelry images
   - Recommended size: 1200x1200px or larger
   - Formats: JPG, PNG, WebP

2. **Upload to Sanity:**
   - Go to Sanity Studio (`/studio`)
   - Create Site Settings document
   - Upload hero image, about image
   - Create jewelry designs and upload product images

3. **Image Requirements:**
   - Minimum: 800x800px
   - Recommended: 1200x1200px
   - Maximum file size: 10MB
   - Formats: JPG (preferred), PNG, WebP
